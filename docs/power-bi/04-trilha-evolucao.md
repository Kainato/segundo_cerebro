# Trilha de Evolução

## 🌱 Iniciante

- Interface e navegação
- Importar e tratar dados
- Primeiros gráficos
- Relatórios simples

## 📈 Intermediário

- Modelagem dados
- Funções DAX principais
- Dashboards temáticos
- Power Query avançado

## 🚀 Avançado

- DAX complexo (CALCULATE, FILTER)
- Inteligência temporal
- Soluções corporativas
- Publicação e governança

## 💼 Analista Estratégico

- Pensamento orientado problema
- Storytelling com dados
- Projetos portfólio
- Cases negócio reais

---

## As Viradas Mentalidade

**Aula 1 →** Quem sabe Power BI sem negócio vira operador gráfico. Quem entende negócio vira analista estratégico.

**Aula 2 →** Quem importa dados é iniciante. Quem cria métricas úteis que explicam desempenho vira analista.

**Aula 3 →** Ferramenta + cálculo não geram valor. **Problema + insight + comunicação** = valor real empresa.
