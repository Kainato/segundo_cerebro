# Aula 3 — Dashboard (Solução)

## 💡 Ideia Central

Dashboard começa na pergunta certa — não no Power BI.

## Pensamento Orientado Problema

Antes abrir Power BI:
- Empresa quer saber o quê?
- Qual decisão precisa tomar?
- Qual problema precisa resolver?

✅ Só depois abre Power BI.

## Fluxo Completo Projeto Real

1. **Conectar dados** — Excel, BD, APIs, SharePoint
2. **Tratar dados** — Power Query limpa, padroniza, remove duplicatas
3. **Modelar** — Relacionamentos entre tabelas, schema estrela
4. **Criar medidas DAX** — Receita, margem, KPIs
5. **Construir visualizações** — Gráficos, cartões, tabelas, mapas
6. **Gerar insights** — Padrões, ações sugeridas
7. **Publicar** — Power BI Service para equipe

## Boas Práticas Dashboard

- 🎯 **Clareza** — Evita excesso informação. Foca decisão.
- 👑 **Hierarquia visual** — KPI destaca. Detalhes depois.
- 📐 **Contexto sempre** — Comparações: período, meta, variação %.
- 🎛️ **Interatividade** — Filtros permitem exploração.
- 📱 **Adapte público** — Desktop, mobile, nível técnico.
