# Aula 1 — Visão de Negócio

## 💡 Ideia Central

Power BI não gera gráficos bonitos — gera decisões.

## O que é Business Intelligence

- Transforma dados em decisões
- Gera insights para empresa
- Converte dados brutos em visualizações
- Cria vantagem competitiva

## Por que usar Power BI

- Decisão mais rápida e segura
- Integração com múltiplas fontes
- Visualização clara de complexidade
- Comunicação entre departamentos

## Pipeline Mental do Power BI

```
Coleta (Excel, API, BD)
    ↓
Tratamento (Power Query)
    ↓
Modelagem (Relações + DAX)
    ↓
Visualização (Dashboards)
    ↓
Publicação (Power BI Service)
```

## Power BI Desktop vs Service

### 💻 BI Desktop

Construir tudo — modela, cria relatórios, DAX. Instalado localmente.

### ☁️ Service

Distribuir tudo — publicação, compartilhamento e colaboração nuvem.

## Mentalidade de Negócio

Mudar mindset: ferramenta → resolver problemas dados.

**Perguntas fundamentais:**
- Onde empresa perde dinheiro?
- Qual produto vende mais?
- Qual cliente gera valor?
- Tendência escondida existe?
