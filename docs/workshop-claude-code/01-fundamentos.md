# Aula 1 — Fundamentos e Mentalidade de Builder com IA

## 🧩 Mudança de Paradigma: de Usuário para Criador

A primeira aula estabelece uma **mudança fundamental** em como você interage com IA:

| Transição | Descrição |
| --- | --- |
| **De** | Usar IA apenas para respostas rápidas |
| **Para** | Construir sistemas que trabalham por você |

A IA deixa de ser uma ferramenta pontual e passa a ser um **ativo produtivo**, integrado ao fluxo de trabalho.

---

## 🤖 O que é o Claude Code na Prática

O Claude Code não é apenas um chat — ele é um agente autônomo de desenvolvimento, capaz de:

Ler e entender projetos inteiros
Editar arquivos automaticamente
Executar comandos no terminal
Trabalhar com múltiplos arquivos e contextos

Ou seja, ele atua como um “programador assistente com autonomia operacional”.

## 🧠 Conceito Central: IA como Sistema, não como Prompt

Um dos principais aprendizados da aula:

O valor não está no prompt isolado, mas no sistema que você constrói ao redor da IA.

Isso inclui:

- Contexto (dados, histórico, arquivos)
- Estrutura (fluxos e regras)
- Objetivo claro (problema real a resolver)

## 🏗️ Introdução ao “Segundo Cérebro”

A ideia central do projeto do workshop:

Um sistema que:

1. Armazena conhecimento (documentos, experiências, dados)
2. Processa essas informações com IA
3. Gera insights e automatizações

### Exemplos práticos

1. Marketing: análise de campanhas e geração de novos testes
2. Produto: apoio em discovery e validação
3. Operações: consolidação de relatórios automáticos

## ⚙️ Primeiro Contato com Construção

Na Aula 1, inicia-se a construção da ferramenta:

- Definição do problema a ser resolvido
- Estrutura inicial do projeto
- Primeiras interações com IA de forma orientada

O foco não é tecnologia, mas clareza de propósito.
